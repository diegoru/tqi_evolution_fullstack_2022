# Bootcamp TQI Fullstack Developer



## Projeto Dbook Livraria



A ideia foi desenvolver uma API  RestFull com Spring Boot Java para ser consumida pelo React. 

**Back-end:**



- Spring Boot 3 com Java 18

- Banco de Dados SQL
- Hibernate JPA
- Migrations com Flyway
- Padrão VO
- Swagger
- JWT 
- Spring Security
- Junit
- Mockito



**O Front-end :**

- React
